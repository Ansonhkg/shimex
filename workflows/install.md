# Install Workflow

Trigger: `shimex install`

1. Detect the user's source Codex Desktop app.
2. Read source version/build metadata.
3. Plan a managed Shimex app copy.
4. Ask for approval before app bundle writes.
5. Copy the source app to the managed Shimex app path.
6. Write isolated Shimex Codex profile config.
7. Start or restart the Shimex server.
8. Open the managed Shimex app.

The original Codex app and normal Codex profile are never modified.
